# Automated Expense Tracker

## About the Project

I created this Automated Expense Tracker using Python to help users keep track of their daily expenses and manage their monthly budget.

The program allows users to add expenses, view their spending history, calculate total expenses, set a monthly budget, and generate visual reports. I developed this project to strengthen my understanding of Python programming, file handling, data analysis, and data visualization.

## Features

- Add and save expenses
- View all recorded expenses
- Calculate total spending
- Set a monthly budget
- Get notified when spending exceeds the budget
- Generate expense reports by category
- Create pie chart visualizations of spending patterns

## Technologies Used

- Python
- Pandas
- Matplotlib
- JSON
- Jupyter Notebook

## Project Files

```text
expense_tracker.ipynb   # Main notebook
expenses.json           # Stores expense data
expense_report.png      # Generated report chart
README.md
requirements.txt
```

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Open Jupyter Notebook:

```bash
jupyter notebook
```

3. Open the notebook and run all cells.
